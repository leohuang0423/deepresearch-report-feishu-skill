# Native Dossier Rules

Use this module for the source-of-truth local report package.

## Goal

Create a folder structure that is easy for humans, AI systems, and agents to inspect.

## Expected Contents

- main report
- appendix
- atomic evidence ledger
- case packets
- figure manifest
- render manifest
- open questions

## Rule

The dossier is the canonical source. Feishu and HTML outputs are downstream render targets.
