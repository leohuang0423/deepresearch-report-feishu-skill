# Reader-First Narrative Rules

Use this module when turning research into a readable report.

## Opening Standard

In the first 12-15% of the report, make these explicit:

- audience
- decision goal
- thesis
- why this matters now
- scope boundary

## Section Pattern

Every major section should follow:

1. question
2. answer
3. evidence
4. implication

## Writing Rules

- lead with the answer, not with scene-setting
- prefer pyramid structure over timeline dumping
- compress repetition aggressively
- write transitions that help a busy reader decide whether to keep reading

## Failure Smells

- reads like stitched research notes
- main point arrives too late
- elegant prose but weak decision utility
- AI-slop cadence and generic throat-clearing
