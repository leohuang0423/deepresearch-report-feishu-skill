# Report Visual Rules

Use this module when the report needs publication-grade figures.

## Principle

Figures are part of the reasoning chain, not decoration.

## Draft vs Publish

- internal drafts may use rough diagrams
- final publish assets must not rely on Mermaid as the final visual output
- publish figures should be rendered as SVG, PNG, or HTML views from structured specs

## Required Figure Roles

Use at least one figure that clarifies the core judgment, and at least one figure that compresses comparison, flow, or sequence.

## Preferred Figure Types

- decision chart
- mechanism/system diagram
- comparison/timeline
- case flow breakdown

## Quality Checks

- if the figure is harder to read than the paragraph it replaces, it failed
- if the figure adds no decision value, cut it
- keep typography, spacing, and labeling consistent across figures
