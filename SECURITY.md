# Security Policy

## English

Treat third-party skills as executable operational instructions.

This repository has a strict public release policy:

- no local machine paths
- no hidden install dependencies outside the published skill folder
- no bundled secrets
- no internal debug traces

If you discover a security issue:

- do not open a public issue with exploit details
- email or privately contact the maintainer first
- include the affected path, reproduction steps, and expected impact

## 中文

请把第三方 skill 当作“可执行的操作说明”来看待。

这个仓库对公开发行有严格要求：

- 不允许本机路径泄露
- 不允许安装后依赖仓库外部隐藏路径
- 不允许打包任何 secret
- 不允许公开内部 debug trace

如果你发现安全问题：

- 不要先公开提交带 exploit 细节的 issue
- 先私下联系维护者
- 说明受影响路径、复现步骤和影响范围
